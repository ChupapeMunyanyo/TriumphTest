# TinyMCE Editor

Простой редактор с поддержкой:
- Форматирования текста
- Вставки кастомных компонентов
- Адаптивным интерфейсом

## Использование
1. Откройте `index.html`
2. Начните редактирование

Требуется API-ключ TinyMCE Cloud.
